# TaskFlow

**TaskFlow** es una aplicación web moderna para la gestión de tareas y
proyectos basada en tableros Kanban. Permite organizar el trabajo de
forma visual, flexible y eficiente, con una experiencia de usuario
fluida y completamente responsive.

## Características Principales

-   Tableros Kanban dinámicos
-   Gestión avanzada de tareas con prioridades
-   Interfaz Drag & Drop
-   Modo claro y oscuro automático
-   Portadas personalizadas
-   Persistencia con LocalStorage
-   Diseño responsive
-   Alto rendimiento con React Hooks

------------------------------------------------------------------------

## Instalación

``` bash
git clone https://github.com/Antonio-Benavente/TaskFlow.git
cd taskflow
npm install
npm run dev
```

------------------------------------------------------------------------

## Build para Producción

``` bash
npm run build
```
